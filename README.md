Наполнение базы данных по ученикам (файл .json):
1. ФИО
2. Дата рождения
3. Класс
4. ФИО классного руководителя
5. Адрес
6. Номер телефона родителя
7. Средняя успеваемость

Графичкский интерфейс выполнен через библиотеку easygui.
